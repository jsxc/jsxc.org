JSXC Feature List

  • Full-featured XMPP client in the browser, no client installation required
  • Responsive design
  • Easy integration into your existing web page or application
  • Interaction with your web page, e.g. making Jabber IDs active
  • Uninterrupted browsing among JSXC-enhanced pages
  • One-to-one chats, optionally with end-to-end encryption (OTR) and user verification (SMP)
  • End-to-end encrypted SRTP/WebRTC audio and video calls from Firefox and Chrome without plugin
  • multi-user chat (MUC) rooms for text messages
  • file transfer via WebRTC
  • A growing number of international languages supported (add more)
  • Event notifications with sound if browser/tab inactive (not yet supported in Internet Explorer)
  • Roster management
  • Support for vCard avatars
  • Emoticons
  • Link/E-Mail/URI scheme detection
  • Chat synchronized between multiple tabs (not yet supported in Internet Explorer)

Supported protocols

  • XMPP Core (RFC6120)
  • XMPP IM (RFC6121)
  • Multi-user chat (XEP-0045)
  • Result set manageent (XEP-0059)
  • Bidirectional-streams Over Synchronous HTTP (XEP-0124)
  • XMPP Over BOSH (XEP-0206)
  • Service Discovery (XEP-0030)
  • CAP (XEP-0127)
  • Jingle (XEP-0166)
  • Jingle RTP Sessions (XEP-0167)
  • Jingle File Transfer (XEP-0234)
  • vcard-temp (XEP-0054)
  • vCard-Based Avatars (XEP-0153)
  • Message Delivery Receipts (XEP-0184)
  • URI Scheme Query (XEP-0147)
  • Message Carbons (XEP-0280)
  • Stanza Forwarding (XEP-0297)

Supported browsers

  • Full support for Chrome and Firefox
  • Safari does not yet support WebRTC (audio and video chats)
  • Internet Explorer does not yet support multiple tabs, WebRTC, and notifications

Planned features

  • Multi User Chat (MUC) room configuration
  • Audio/video conference
  • Encrypted file transfer
